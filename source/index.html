推荐资源整理

Sat, Feb 25, 2017

整理了一些工作中可能会经常用到的提升工作效率的资源，包括前端和一些效率工具。

前端

1. Materializecss

基于Google Material Design的现代化响应式前端框架http://materializecss.com/

2. React

A declarative, efficient, and flexible JavaScript library for building user interfaces. https://github.com/facebook/react

3. Mustard-UI

一个压缩之后只有5k左右的纯css框架. https://mustard-ui.com/

Nodejs

1. Hexo

A fast, simple & powerful blog framework. https://hexo.io/

2. Koajs

Next generation web framework for Node.js. https://github.com/koajs

3. FPM

适合初创技术团队使用的轻量级前后端分离框架. https://github.com/team4yf/yf-fpm-server

4. Bluebird

Tools

1. lodash

2. dayjs

一个可以替代 moment.js 的小型日期处理库

3. axios

Resource

1. Iconfinder

找icon的好地方 https://www.iconfinder.com/

2. 全景照片素材

最近在研究vr，需要一些前景照片，这里有很多素材供选择 https://www.flickr.com/search/?text=equirectangular

3. Guides for fe-jser

前端学习手册，有丰富的教学文档，类似于笔记，非常实用 https://github.com/pluralsight/guides/tree/master/published/front-end-javascript 当然你也可以看它提供的视频教学(部分收费) https://www.pluralsight.com/

4. WebVR

基于浏览器的vr技术资料

  • 一个正在学习vr技术的geeker的笔记 webvr-experiments
  • 非常酷的vr框架 aframe
    • 看一眼案例就很炫酷了

5. 常规数据

一些开发过程中常用的数据，比如城市街道、汽车品牌等。

  • 汽车数据
    • 品牌
    • 车型
    • 年份
    • 指导价格
    • logo

数据已经打包成sql语句[mysql] 、data.json 和 图片压缩包。下载地址：http://pan.baidu.com/s/1dESFSJJ

6. Oracle

下载Oracle的产品

体验很棒！ 比如：jdk, oracle数据库, weblogic server 等等。 https://edelivery.oracle.com/osdc/faces/SearchSoftware
